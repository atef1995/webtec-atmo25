Redovisning

Här kommer mina redovisningstexter från de olika kursmomenten i denna kursen.

kmom01

Min utvecklingsmiljö för denna kurs består av Visual Studio Code som huvudsaklig editor, Git för versionshantering och Node.js för att köra utvecklingsverktyg som ESLint och http-server. Jag har också installerat npm-paket som är nödvändiga för kursen enligt instruktionerna. Utvecklingsmiljön känns bekant och stabil, vilket gör det enkelt att fokusera på själva kodandet istället för att krångla med verktyg.

Installationen av kursens utvecklingsmiljö gick relativt smidigt. Jag hade redan Git och Node.js installerat sedan tidigare, så det var mest en fråga om att klona kursrepot och installera npm-dependencies. ESLint-konfigurationen var lite ny för mig, men efter att ha läst dokumentationen förstod jag hur den fungerar och varför den är användbar för att hålla konsistent kodkvalitet.

Jag har jobbat en del med terminalen tidigare, framförallt grundläggande Unix-kommandon som ls, cd, mkdir och git-kommandon. Det känns naturligt att navigera i filsystemet via kommandoraden och att använda Git för versionshantering. Terminalen är definitivt ett kraftfullt verktyg när man väl lärt sig grunderna.

HTML, CSS och JavaScript är tekniker jag stött på tidigare, men mest på en grundläggande nivå. Jag har byggt enkla webbsidor tidigare, men inte med den struktur och organisation som vi använder i denna kurs. Det var intressant att se hur man kan strukturera ett projekt med separata mappar för CSS och JavaScript, och hur man kan skapa en konsistent design med återanvändbara komponenter som header och footer.

Att komma igång med kursmomentet var överlag ganska rakt fram. Den stegvisa övningen hjälpte mycket att förstå hur alla delar hänger ihop. Det som var lite utmanande var att förstå CSS-layouten med grid för två-kolumners design, men efter lite experimenterande började det klarna. JavaScript-delen för att sätta aktuellt år var enkel men smart - det visar hur man kan använda JavaScript för praktiska funktioner.

Min TIL för detta kursmoment är hur kraftfullt aside och main är för att skapa layouter. Tidigare har jag mest använt divs för positionering
