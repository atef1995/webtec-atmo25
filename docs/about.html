Om kursen webtec

Kursen webtec är en grundkurs i webbutveckling som fokuserar på de fundamentala teknologierna för att skapa moderna webbplatser. Kursen täcker HTML för struktur, CSS för styling och layout samt JavaScript för interaktivitet och dynamisk funktionalitet.

Genom kursen lär vi oss att bygga responsiva webbplatser som fungerar bra på alla enheter, från mobiler till stationära datorer. Vi arbetar också med utvecklingsverktyg som Git för versionshantering, linters för kodkvalitet och moderna arbetssätt inom webbutveckling.

Kursrepo och material

Allt kursmaterial och exempel finns tillgängligt på kursens officiella repository på GitHub: dbwebb-se/webtec. Här hittar du övningar, uppgifter och exempel som används genom hela kursen.

Kursmål och innehåll

Efter genomgången kurs ska studenten kunna utveckla välstrukturerade webbplatser med ren kod som följer web standards. Kursen betonar vikten av semantisk HTML, effektiv CSS och väl organiserad JavaScript-kod.
